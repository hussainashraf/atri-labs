
# Setup project Locally

A brief description how to stup project locally

## Installation

pip install --user pipenv


1. Create a project directory
```bash
  mkdir my_app
  cd my_app
```
2. Install atri CLI
```bash
  pipenv install atri==0.0.7
```
3. Activate virtual environment
```bash
  pipenv shell
```
4. Start Atri engine
```bash
  atri start
```
